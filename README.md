# 🍱 씨마스 급식 (C-MAS Lunch)

씨마스고등학교 학생들을 위한 **고성능 웹/앱 하이브리드 급식 정보 서비스**입니다. React와 Tailwind CSS를 기반으로 제작되었으며, 모바일 앱과 같은 부드러운 사용자 경험을 제공합니다.

## ✨ 주요 기능

-   **오늘의 식단 & 주간 식단**: 고화질 메뉴 이미지와 함께 영양 성분이 포함된 상세 식단을 확인하세요.
-   **실시간 급식 톡**: 친구들이 남긴 생생한 별점과 후기를 실시간으로 확인할 수 있습니다.
-   **알레르기 설정**: 본인의 알레르기 정보를 설정하면 식단 확인 시 안전하게 필터링할 수 있습니다.
-   **PWA 최적화**: 스마트폰 홈 화면에 추가하여 설치된 앱처럼 사용할 수 있습니다.
-   **반응형 레이아웃**: 데스크톱 웹과 모바일 앱 환경 모두에 최적화된 고급스러운 UI를 즐겨보세요.

## 🛠 사용된 기술 (Tech Stack)

-   **Frontend**: React 18, TypeScript, Tailwind CSS 4
-   **Animation**: Motion (framer-motion)
-   **Icons**: Lucide React
-   **Architecture**: Mobile-First Hybrid Layout

## 🚀 시작하기

### 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# 프리뷰 실행
npm run preview
```

## 📱 앱 설치 방법 (PWA)

1.  모바일 브라우저(Safari, Chrome)에서 서비스 접속
2.  브라우저 하단의 **'공유'** 또는 **'설정'** 버튼 클릭
3.  **'홈 화면에 추가'** 선택
4.  바탕화면의 앱 아이콘을 통해 즉시 실행
